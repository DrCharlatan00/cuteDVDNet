# What does this program do? 
This program allows you to view files on CD/DVD discs, as well as download them.

# How do I use this? 
Download the release package and run it. Alternatively, you can build it from source:
1. Download the source code
2. Navigate to the `/cuteDVDNet` directory
3. Run `dotnet run -c Release`

You can also download a UI shell (interface) from this link: [-> click <-](https://github.com/DrCharlatan00/cuteDVDDesktop) or create your own.
